Welcome to ComboTweet!

What is it?

  • An AJAX-powered Twitter client that allows for logging in to multiple accounts simultaneously. Learn more

Options for using ComboTweet:

  • On the web

    Simply visit http://combotweet.com/client and start using it!

  • Run it locally

    If you run a web server on your computer that can execute PHP (such as MAMP for Mac OS X), you can download the source for ComboTweet and run it on your own machine.

    Simply unzip the source into a directory that your server has access to and visit the address in your browser. For example, in MAMP, you could unzip combotweet_src.tgz into /Applications/MAMP/htdocs, and then visit localhost:8888/combotweet/ and click on "Start using it" at the top of this page

  • As an Adobe AIR application

    - Currently Unavailable - will be back soon

    There is an Alpha version of the ComboTweet client for Adobe AIR available here. Download the AIR package and double click to install.
