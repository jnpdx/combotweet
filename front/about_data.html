Welcome to ComboTweet

What is it?

  • An AJAX-powered Twitter client that allows for logging in to multiple accounts simultaneously.

Who made it?

  • John Nastos - @jnpdx

Why is it hosted at Tagalus?

  • ComboTweet integrates with Tagalus, a hashtag definition service. Hashtags in tweets displayed in ComboTweet are automatically linked into Tagalus

What can it do?

  • Login to multiple Twitter accounts at the same time
  • Auto-refreshes tweets
  • Integrated hashtag searching with Tagalus
  • Twitter search (results from search.twitter.com)

How do I use it?

  • Buttons:
    • Add panel
      Open a new panel to login with a Twitter username and password. If a login panel is already open, this closes it.
    • Refresh
      Refresh the current panel
    • Search
      Search Twitter for a given term or set of terms.
    • Settings
      The settings give you access to a few options that ComboTweet has
      • Tabbed Browsing: If true, panels are displayed one-at-a-time and navigated through via tabs in the header. If false, all panels are displayed all the time, side-by-side.
      • Remove Old Tweetgs:If true, tweets at the bottom of a panel are removed as new tweets come in
      • Append hashtag: If true, there is an extra input near the "Update" button that allows you to enter a tag. If there is a tag present, each tweet sent will have the tag appended it, until the tag is cleared, or the option is turned off
      • Refresh Frequency: Number of seconds between refreshes. Numbers below 45 (for extended periods of time) are more likely to push you over Twitter's API limit, and thus be ineffective
      • Font Size: The size of the font in the displayed tweets
      • Panel Width: The width of the panel. Having a larger width can be nice when only viewing one or two panels, while smaller widths may be convenient when many panels are open. Values below 400 are not allowed
    • Close
      Clicking on a this icon on a tab closes a singular panel. The larger version of this button next to the Search icon closes all open panels
